This plugin allow user to rate the post/page by adding star rating control to the existing comment feature of Wordpress.

== Description ==

- Allow user to choose what post/page to be allowed to rate.
- Allow user to choose that categories to be allowed to rate.

== Installation ==

1. Upload `post-review` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('comment_form_rating_field'); ?>` in your templates
